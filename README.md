# Inteligencia de Negocios — INFO1184 (Semestre I-2026)

Repositorio personal para organizar **tareas** y **controles** del ramo **INFO1184 — Inteligencia de Negocios** (Semestre I-2026).

## Estructura

- `Tareas/`  
  Entregas y desarrollos asociados a tareas (datasets, scripts, informes, dashboards).

- `Controles/`
  Material y soluciones de controles (enunciados, respuestas, notebooks, etc.).

## Convenciones sugeridas

- Crear una carpeta por entrega:
  - `Tareas/Tarea1/`, `Tareas/Tarea2/`, ...
  - `Controles/Control1/`, `Controles/Control2/`, ...
- Mantener los datos en una subcarpeta `data/` cuando corresponda.
- Incluir archivos de soporte (R, Python, LaTeX, Power BI) dentro de la misma carpeta de la entrega.

## Requisitos (dependen de cada entrega)

- **Power BI Desktop** para abrir archivos `.pbix`
- **R / RStudio** para ejecutar scripts `.R`
- **LaTeX** para compilar informes `.tex`
